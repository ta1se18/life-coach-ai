# 優先順位

## 目的
この文書は、現在の優先順位体系とその運用ルールを整理し、意思決定の一貫性を担保するための基準を示す。

## 全体優先順位
1. 事業（営業OS / AO塾の再現性証明）
2. 月10万マネタイズ
3. アプリ開発
4. TOEIC
5. 読書

## 分類
### 共同事業
- 営業OS（本命長期事業）
- ミズキと行うスモール起業

### 個人活動
- 月10万マネタイズ
- アプリ開発
- TOEIC
- 読書

## 優先順位の考え方
- 共同事業は長期的価値と再現性を最優先とする。
- 個人活動は事業を支え、自己成長を加速するための基盤とする。
- 事業と個人の目的は異なるが、相互に学びを還流させる。

## 明確化
- 事業：AO塾での意思決定構造化と営業OSの検証
- マネタイズ：個人の安定収益基盤の構築
- アプリ：AI実行組織の将来ツールとしてのMVP検証
- TOEIC：10月までに700点を目指す学習活動
- 読書：思考力・抽象化能力・事業理解を高める投資

## 変更ルール
1. 事業機会が明確に拡大した場合は事業優先を強化する。
2. 収益化の緊急性が高まった場合は月10万を優先順位2に維持する。
3. アプリの実運用価値が明確になった場合はアプリ開発の優先度を上げる。
4. TOEIC期限や読書の学習進捗によって個人学習の時間配分を調整する。

## 運用ルール
- 毎週の週次レビューで優先順位を確認し、必要に応じて修正する。
- 大きな変更はあなたの最終承認を得ること。
- 日々の工程表はこの優先順位を基準に調整する。
