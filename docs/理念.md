# 理念

## 目的
この文書は、AI実行組織と共同事業の長期運用における根本的な価値観と方向性を記録する。

## 事業の本質
- 人間の意思決定変化を対話から構造化し、再現可能な知識へ変える。
- 営業は入口であり、目的は“人間理解”と“意思決定設計”の再現性である。
- 最大の資産はログと仮説の蓄積であり、AIはそれを補助するための道具である。

## 長期ビジョン
- AO塾を起点として、営業現場の意思決定変化を再現可能にする。
- 共同事業では「意思決定の構造化」を証明し、後のSaaS化につなげる。
- 個人活動では、月10万の収益基盤とTOEIC700点、思考力の強化を通じて事業を支える。

## 価値観
- 思考のスピードを落とさず、日本語の解像度を活かして深い人間理解を行う。
- 事業と個人は分離しつつ、互いの学びを還流させる。
- まずは動くことを優先し、実装は運用と検証の後に進める。

## 望む状態
- AIが文脈を忘れず、継続的に成長できる運用基盤ができている。
- 共同事業の再現性と個人の収益基盤の両輪が回っている。
- 長期的に「人間理解」を中心とした意思決定OSを目指す。
