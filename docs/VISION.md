# VISION

## 目的
この文書は、現在の事業と個人活動を包括的に整理し、AIを中心とした長期運用の基盤を作るためのビジョンを記録する。

## 事業の最終ビジョン
- **人間の意思決定変化を対話から構造化し、再現可能な知識へ変える**ことを核とする。
- 営業は入口であり、目的は「顧客理解」「意思決定設計」「人間理解」の再現性である。
- AIはデータと構造化の補助役であり、最大の資産は「対話ログと仮説の蓄積」である。

## 共同事業のビジョン
- **共同事業（営業OS / AO塾 / ミズキとのスモール起業群）**は、複数の現場で意思決定構造を記録し、再現性を検証する場となる。
- AO塾を最初の再現性検証市場とし、対話から決定要因を抽出する仕組みを実証する。
- 検証結果を元に、将来的には営業AI・意思決定分析OSを提供する。

## 個人活動のビジョン
- **個人活動（個人収益 / アプリ開発 / TOEIC / 読書）**は、事業成長と自己成長を支えるパーソナル基盤である。
- 収益化は「月10万を安定的に稼ぐ仕組み」を目標とし、事業の学びを個人の力に変える。
- アプリは将来的にAI実行組織を支えるツールであり、今は運用を優先した検証段階とする。
- TOEICと読書は、グローバルと抽象化能力を高める投資であり、事業の思考基盤になる。

## 成長の前提
- AIが文脈を忘れず、長期的に成長するためには、**一貫したドキュメントと運用ルール**が必要。
- 毎朝の工程表・週次レビュー・月次戦略で、常に現状と目標を整合させる。
- 共同事業と個人活動は分離しつつ、相互に学びとリソースを循環させる。

## 望む状態
- 事業と個人の両輪が共鳴し、AI運用組織として継続的に改善できる状態。
- 共同事業ではAO塾を起点に再現性を確立し、個人では月10万の収益とTOEIC700点を目標に成長する。
- 5年後には、「人間理解」をベースにした意思決定OSが市場に存在する。
